# 帮助我们改善文档

  1. 发起issue (opens new window)

  2. 发起PR

    1. 文档基于 vuepress doc (opens new window), 建议先大致阅览一遍vuepres文档。
    2. Fork该仓库。
    3. 找到需要修改的文档，更新文档。
    4. 提交代码，带上详细的commit信息或者相关issue编号
    5. 发起PR (opens new window)

# 文档阅览方式

  1. 确保本地有node环境
  2. 在项目根目录执行：yarn/npm install
  3. 阅览文档：yarn/npm run docs:dev
